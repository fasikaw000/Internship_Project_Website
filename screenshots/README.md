## How to Capture Screenshots

### For Web Pages:
1. Run both frontend and backend servers
2. Navigate to each page in your browser
3. Use browser's screenshot tool or press `Windows + Shift + S` to capture
4. Save with the exact filename listed above

### For Email Screenshots:
1. Check your email inbox for the automated emails
2. Open each email
3. Capture screenshot of the email content
4. Save with the appropriate filename

### For Mobile Views:
1. Use browser's developer tools (F12)
2. Toggle device toolbar (Ctrl + Shift + M)
3. Select a mobile device (e.g., iPhone 12 Pro)
4. Navigate to pages and capture screenshots

## Tips for Good Screenshots:
- Use full-page screenshots when possible
- Ensure text is readable
- Include relevant UI elements
- Use consistent browser window size for desktop screenshots
- Remove any sensitive personal information before capturing

Once all screenshots are captured and saved in this directory, the Part Four documentation will be complete with visual evidence of your project.
